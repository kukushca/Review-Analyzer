# Анализатор отзывов о ресторане с дашбордом

Скрипт на Python для автоматического контент-анализа текстовых отзывов о ресторане с использованием большой языковой модели DeepSeek через OpenAI API. Программа группирует отзывы по ключевым категориям, оценивает их по 10-балльной шкале и строит интерактивный графический дашборд аналитики.

## Функционал
* Интеллектуальный анализ: Извлечение оценок и категорий («Кухня», «Обслуживание», «Интерьер», «Цены») из сырого текста отзывов.
* Логирование: Полное протоколирование работы и автоматический перехват ошибок (FileNotFoundError, сбои сети, некорректные ответы ИИ) с записью в файлы логов.
* Визуализация: Построение дашборда с круговой диаграммой долей оценок и столбчатой диаграммой средних баллов по критериям в одном окне.
* Тестирование: Проект полностью покрыт автоматическими тестами на фреймворке pytest.

## Архитектура проекта
* main.py — основной скрипт программы (чтение данных, запросы к API, визуализация).
* test_main.py — модульные тесты для проверки стабильности функций и обработки ошибок.
* reviews.csv — файл с исходными текстовыми отзывами (по одному отзыву на строку).
* config.txt — конфигурационный файл для хранения API-ключа.

## Запуск проекта

1. Подготовка окружения и зависимостей
Убедитесь, что у вас активировано виртуальное окружение venv, и установите необходимые библиотеки через терминал:
```bash
pip install openai matplotlib pytest
```
2. Настройка API-ключа
Создайте в корне проекта файл config.txt и добавьте туда ваш ключ в следующем формате (без пробелов):
DEEPSEEK_API_KEY=ваш_реальный_api_ключ

3. Запуск анализа
Выполните в терминале команду:
```bash
python3 main.py
```


4. Запуск тестов
Для проверки работоспособности модулей выполните команду:
```bash
pytest test_main.py -v
```
